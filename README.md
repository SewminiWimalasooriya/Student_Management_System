
# 🎓 Student Management System

A full-stack web application to manage student records efficiently. Built using TypeScript, Node.js, MongoDB, and Vite.

---

## 🚀 Features
- Add, update, and delete student records
- View student details
- Responsive UI
- REST API integration

---

## 🛠️ Tech Stack
- Frontend: Vite + TypeScript
- Backend: Node.js + Express
- Database: MongoDB

---

## 📂 Project Structure

project-root/
├── Frontend 
├── Backend 
└── README.md

## Install dependencies

cd frontend
npm install

cd backend
npm install

## Run the project

# Run backend
cd frontend
npm start

# Run frontend
cd backend
npm run dev
